# fastboot生产级快速开发脚手架
**极简.规范.优雅**


fastboot是一套围绕着springboot技术栈的脚手架系统，用于快速开发基于spring的web项目

目录结构：

1、fb-common 公共工具

2、fb-frame 框架层

3、fb-generator 代码生成器

4、fb-plugs 插件包

       multipleds 多数据源组件
   
       security 安全组件
   
5、fb-web web脚手架

6、fb-kotlin-web web脚手架（kotlin版本）


后端：

        springboot 2.5.0
        mybatis-plus 3.4.2
        druid 1.2.6
前端：

        vite
        vue3
        antd
